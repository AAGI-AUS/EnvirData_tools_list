# EnvirData_tools_list

A simple repo sharing a list of environmental data (*e.g.*, climate and soils) acquisition tools 

## R Packages

### Weather Data

- [{GSODR}](https://docs.ropensci.org/GSODR/)
- [{nasapower}](https://docs.ropensci.org/nasapower/)
- [{weatherOz}](https://docs.ropensci.org/weatherOz/)
- [{weathervane}](https://biometryhub.github.io/weathervane/)
 
## Soil Data

- [{SLGACloud}](https://github.com/AusSoilsDSM/SLGACloud)
- [{read.abares}](https://codeberg.org/adamhsparks/read.abares)
  - [Soil Thickness Data](https://adamhsparks.codeberg.page/read.abares/reference/get_soil_thickness.html)
